## **단축키 Tip**    
- Ctrl + space 자동완성     
- Ctrl + F11 실행    
- Ctrl + d 한 줄 삭제     
- Ctrl + Shift + +/- 글자 크기 조절    
- syso + Ctrl + Space 출력 코드 자동 완성    
- 들여쓰기는 Tab키로 (= Space 4칸)    
- Shift + Tab 들여쓰기 지우기    
- 기본생성자 : Ctrl + Space + Enter    
- 생성자 자동 생성 단축키: Alt + Shift + S -> O    
- Getter / Setter 자동 생성 단축키 : Alt + Shift + S -> R
